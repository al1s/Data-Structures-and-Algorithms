# Single Linked List

Implementation of a single linked list

## Challenge

Implement linked list with the following methods and efficiency:

- Constructor to require at least one Node;
- Add (O(1));
- Print (O(n));
- Append(O(n));
- AddBefore(O(n));
- AddAfter(O(n));
